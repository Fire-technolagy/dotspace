🤯 What Is This?

DotSpace is a Turing-complete programming language that uses ONLY two characters: a period (.) and a blank space.

No letters. No numbers. No symbols. Just dots and spaces.

And somehow, it can compute anything. 🤖

⚡ Install in 4 Commands

Terminal
git clone https://github.com/Fire-technolagy/dotspace.git cd dotspace chmod +x install.sh ./install.sh

Then run any DotSpace file:

dotspace hello.ds

📖 The Entire Language

Only 8 commands. That's the whole language.

Dots Equivalent What It Does
.>Move pointer right
..<Move pointer left
...+Add 1 to memory
....-Subtract 1 from memory
......Print as ASCII character
......,Read input
.......[Start loop
........]End loop
