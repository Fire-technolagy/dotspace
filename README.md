## 🤯 What is this?

DotSpace is a **Turing-complete programming language** that uses only TWO characters:

- `.` (a period)
- ` ` (a blank space)

That's it. No letters. No numbers. No symbols. Just dots and spaces.

And somehow, **it can compute anything any other language can.** 🤖

## ⚡ Quick Install

    git clone https://github.com/Fire-technolagy/dotspace.git
    cd dotspace
    chmod +x install.sh
    ./install.sh

Then run any DotSpace file:

    dotspace hello.ds

## 📖 The Entire Language (8 Commands)

| Dots | Command | What it does |
|:---:|:---:|:---|
| `.` | `>` | Move pointer right |
| `..` | `<` | Move pointer left |
| `...` | `+` | Add 1 to memory |
| `....` | `-` | Subtract 1 from memory |
| `.....` | `.` | Print as ASCII character |
| `......` | `,` | Read input |
| `.......` | `[` | Start loop |
| `........` | `]` | End loop |

Separate commands with spaces or newlines. That's the whole language. 🤷

## 🟢 The Spirit of DotSpace

DotSpace programs should be **written by hand**.

Using generators or translators to convert text into dots **misses the entire point**. The art is in the constraint. The challenge is in counting dots. The glory is in suffering through it.

If you use a tool to write your DotSpace code, you are not a DotSpace programmer — you are a Python programmer with extra steps. 😤

**True DotSpace:** Counting dots manually. Suffering. Persevering. Glory. 💪  
**Fake DotSpace:** `python3 generate.py "hello"` ❌

The dots must come from the heart. 🟢

## ⚖️ The DotSpace Law

DotSpace has ONE sacred rule:

> **Only `.` and blanks are allowed. EVER.**

If you fork DotSpace and add any new symbol, congratulations — you made a different language! Call it something else. 😤

This is what makes DotSpace, DotSpace. **The constraint IS the language.**

### What forks CAN do:
- ✅ Speed up the interpreter
- ✅ Add better error messages  
- ✅ Port it to other languages (C, Rust, JavaScript)
- ✅ Build playgrounds and debuggers
- ✅ Write hand-coded example programs

### What forks CANNOT do (and still be DotSpace):
- ❌ Add new symbols (tabs, brackets, anything)
- ❌ Change what the dot counts mean
- ❌ Add a generator that converts text to dots
- ❌ Break the 2-character rule

**The minimalism IS the magic.** 🪄

## 🍴 Naming Your Fork

If you add features beyond the rules above, please rename your project!

Examples of acceptable fork names:
- ✅ DotSpace-Fast (just optimized, same syntax)
- ✅ DotSpace-JS (ported to JavaScript, same syntax)  
- ✅ DotSpace-Web (online playground, same syntax)

Examples that should be renamed:
- ❌ DotSpace 2.0 with new symbols → Call it "TriCode" or something
- ❌ DotSpace with tabs → Call it "DotTab"
